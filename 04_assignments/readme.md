# 🗂️ SQL Mini Project 1 - SmartMart Retail Sales Analysis

## 📌 Overview
This project focuses on analyzing retail sales data using SQL.

The goal is to extract meaningful insights related to:
- Product performance
- Customer distribution
- Sales trends
- Category-wise analysis

---

## 🛠️ Tools Used
- SQL (MySQL)

---

## 🗃️ Database Structure

### 1. Customers Table
- customer_id
- customer_name
- city
- age

### 2. Products Table
- product_id
- product_name
- category
- price

### 3. Sales Table
- order_id
- customer_id
- product_id
- quantity
- order_date

- Basic SELECT queries  
- Filtering using WHERE  
- Sorting (ORDER BY)  
- Aggregation functions (COUNT, SUM, AVG, MAX)

- HAVING clause  
- JOIN operations  
- Business query analysis
