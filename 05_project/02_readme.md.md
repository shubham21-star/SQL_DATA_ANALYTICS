## 📌 Project Overview

This project is part of my academic assignment **"SQL Pro – ShopKart E-Commerce Business Analytics"**.  
The goal of this project is to analyze e-commerce business data using SQL and extract meaningful insights related to customers, orders, products, and sales performance.

---

## 🎯 Objectives

- Perform data analysis using SQL queries  
- Understand customer behavior and purchase patterns  
- Identify top-performing products and categories  
- Analyze revenue, profit, and order trends  
- Practice real-world business query scenarios
